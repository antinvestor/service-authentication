# antinvestor_api_tenancy

Dart client library for the Ant Investor Tenancy Service.

Data partitioning, multi-tenancy, access control, service accounts, and OAuth2 client management.

## Installation

```yaml
dependencies:
  antinvestor_api_tenancy: ^2.0.0
```
